发明语言的人

2024-07-06

刚读完《意象的帝国》这本书，作为一本讲稿，讲的是写诗的技术。读来并无醍醐灌顶之感，也没被侃晕，但确实会被激起莫名其妙的创作冲动，而这正是写作类书籍最需要达到的一点。

以下内容是我个人重新整合的读书笔记，必然会在作者审美偏好的基础上，再涂抹一层自己的好恶。诗歌审美就像人们对“客观”的定义，还是相当主观的，特此通知，后果自负。

阅读课的“逆向工程”

翻开任何一本语文书，字比空白多。随机抽查一节语文课，阅读赏析比写作多。往考场一坐，没记得哪一场让写现代诗。所以，你不能指望通过常规的语文教育学会现代诗的写作。实际上，阅读赏析的思路与写作实操的思路也是大相径庭，基本是“创世论”和“进化论”两种世界观。

在分析一篇课文时，我们常常被要求回答文章细节与主旨之间的联系。文章结构在一遍又一遍的分析与附会中，逐渐变得完整甚至严谨，最后化作对作者海马体容量的讴歌。于是，这样的文本分析方式无疑增加了写作的难度，阅读课不是合格的写作课。

请注意，上面的内容并不是讽刺语文老师对作者的写作牵强附会，导致“窗帘就是蓝色的”等老梗的诞生，而是说哪怕是用字如兵、多智近妖的文体家，他们对于文章的细节果真有暴君般的掌控能力，指望他们挥笔立就也是不现实的。“好文章往往是改出来的”，这就是所谓“进化论”的意味。顺带说一句，使用大模型大概也要用这种思路，指望一次性出结果，结果可能是数都算不对。

写作近乎是即兴行为，并不是只有爵士乐爱好者（村上春树等）和超现实主义者（布勒东等）坚持这一观点。亨利·米勒说，“你在打字机前面做的事情，其实和转账差不多。”凯鲁亚克甚至说，“你已经写完的东西便不再修改，主要是为了将你在写作过程本身当中你大脑的实际成果呈交给读者：你用你无法更改的方式，坦承你对事件的想法。嗯，你想想，你有没有听说过，一个人在酒吧里给一大帮人讲一个又长又离奇的故事，所有人都在笑着听着，你听说过吗，然后这人停下来纠正自己，回到前面一句话，把这句话改得更好，赋予它节奏性的思想影响……”

写作的自己把责任推给修改的自己（凯鲁亚克直接把草稿推给编辑），对写作和修改做一个“课题分离”，这大概是解放创作力的开始。

将熟悉变为陌生

借用鲍曼的一句话，“社会学意味着把熟悉的东西变得陌生，将陌生的东西变得熟悉。”这句话的前一半也是诗人的核心技术。所以古诗天然占优势，就像博物馆里只展览恐龙化石，不放鸡骨头。现代诗歌和你的微信聊天一样生活在白话的水域里，这时，“陌生化”的手一把抓住了“主观意象”这棵稻草。

对于“客观意象”和“主观意象”的划分，可以大致理解为：能看见的是客观意象，想出来的是主观意象。作者在书中一再强调写诗用“意象”而非“概念”，还用书名告诉读者，诗歌是“意象的帝国”。为了达到“一堂课写出好诗句”的 KPI，作者提出了几种看似机械化的策略，协助读者构建主观意象：

  1. 某场景中用错搭模式替换某物
  2. A 是 B
  3. 用 B 解释 A
  4. 让 A 去做 A 做不到的事

你可以拿着作者给的地图，回到曾经读过的诗集里重新寻路，这路大概会走得更明白。就怕走得太明白。

  1. 灯，用门抵住夜的尾巴，窗帘掐紧夜的髦毛，于是在夜宽柔的怀抱，时间便像欢醉的蟋蟀放肆起来。——张枣
  2. 爱是敌意的时差/追赶着死亡的心跳。——北岛
  3. 击鼓之后，我们把在黑暗中跳舞的心脏叫做月亮。——海子
  4. 一架飞机正在飞/飞着飞着就，失去了力量/等待着我们，去营救/我扔出绳子，跟套羊一样/把飞机拉近，再拉近/它越来越小，后来碎在我的手里，就像一块冰。——王敖

主观意象是现代诗中味精一般的存在，“写诗时一旦发觉诗意稀薄，就可以用主观意象来救场”。并且主观意象一般可以抵御翻译带来的磨损，是“全球化”的诗意。此处顺带一提，现代诗特有的换行，或者标点使用带来的停顿，也是“陌生化”的手段之一，今后多留神。

对于“陌生化”这一技术，作者在书中指出：

陌生化甚至可以说，是整个现代文学的看家本领，如果你一时难以把握其精髓，可以采用一种笨办法，来让作品拥有一些陌生感。就是让你的心里，始终绷着一根弦：规避常识。我把它称为“泛陌生化”的意识。既然你一时难以直接构造陌生的意象，那你索性采取排除法，写作时，排除一切你熟悉或知道的说法。比如当你描写落日景象，凡是你熟悉的或学过的那些描述，残霞如血之类，一概排除，这样把常见描述清零之后，你若还能描写落日景象，大概就会有一些陌生感了。当然，你能把常识清零到何种程度，还取决于你知道多少常识，如果你书读得太少，即便你排除了所有你知道的常识，可能有些常识在你心里还完好无损，你一描写，它们就会把描写拉入那些常识套路，陌生感仍难以真正挖掘出来。所以，阅读对写作的帮助，还可以理解为，是为了知道或排除尽可能多的常识。

诗歌创作乃至日常写作中的“陌生化”，要求作者少用成语或大词，把每句话指到实处，“话说得越透亮，才会越有诗意。”（《爱情和其他魔鬼》）这实际上是为发明语言所做的“刻意练习”，所谓文风，正是这训练的结果。

发散与聚合

对于短诗而言，作者认为，“新诗受局部诗意的支配较多，局部难，整体易”。前文把写句子的工作做足了，但胡乱拼接很容易产生“闹剧感”。也就是说，一首诗里意象频出，往往让读者把握不到整体性。

《赖声川的创意学》中讲：

有时找不到答案是因为不知道如何为自己的创意构想设限。创意的自由并非无限的自由。我们要学会设定边线，把构想限制在一个固定范围之内，才会得到创意的自由，不然，无限的发想空间中难以定出结构。

《打造第二大脑》中也说：

“信管法则”的前两个步骤——抓取和组织，构成了“发散”过程。这一阶段的主题是研究、探索和萌发创意。我们会四处收集想象力的种子，并加以妥善保管。后面两个步骤——提炼和表达，则构成了“聚合”过程。这时，我们停止吸纳新思想，并利用收集完毕的知识模块创建新事物。

可见，这是创作的普遍规律。针对现代诗的整体结构，作者也给出了几点建议，可以大致总结为：音乐性、聚合和三段式。

首先，现代诗的韵律远不如古诗的严密，讲押韵的那些，往往只带来了喜剧效果。作者主要抓的是轻重音，可体现为四声的数目，数目少突出空灵感，数目多可以营造出铿锵的效果。进一步，参考“黑山派”的节奏观念，先根据诗歌内容确定合适的情绪，再据此情绪产生的呼吸需要，确定长短句。更简单的方法，则是采用“诗经式”的段间排比，既有音乐性，又容易扣题。

其次，是对陌生的主观意象进行筛选与改造。从诗歌的表达属性出发，主观意象不是越奇怪越好，意象之间还是要有恰当的关联。为了构建这种关联，一来是思考二者的抽象关系，这需要所谓“洞察”，比较像是“浴中奇思”，适合随手记录（前面提到“信管法则”是有原因的）；二来可以构建一个联系两种意象的场景，这也是调动写作时状态的捷径。最重要的是“聚合”，锚定于一个主观意象或主题，进行充分表达。

最后，介绍一个用于“聚合”的三段式结构：$ABA’$，即：第二部分 $B$ 是第一部分 $A$ 的对比或深化，第三部分 $A’$ 是第一部分 $A$ 的重复与扩展。作者认为 $A$ 与 $A’$ 构成了结构上的押韵，这个说法很像我之前的想法：比喻是感受上的谐音。

附录：诗歌接龙

完成一道书中的习题：

找一组人，一起做现场写诗游戏。每六七人或七八人一组，每组合用一张白纸；该组第一人随便写一行文字（简短为好），交给第二人；第二人根据第一人的文字，也写一行文字，把第一人的文字折叠起来（不让第三人看见），再交给第三人；第三人根据第二人的文字，也写一行文字，把第二人的文字折叠起来，交给第四人，以此类推。每组写完后，大声朗读每组合写的文字。读完想一想，为何大家随意写的文字，颇有整体感，甚至像一首完整的诗。

我找了一些大语言模型，按照上面的要求进行创作，并根据个人喜好进行了断句。第一句是布勒东的“自动写作”。描述中所说的“整体感”，是作者用来说明阅读鉴赏“不靠谱”的一个例子，也正是这一点让我联想到大语言模型，“它只是一次添加一个词。”（《这就是ChatGPT》）

诗句及作者如下：

  • 有一个人被窗子劈成了两半（André Breton）
  • 一半在室内凝视着窗外的世界（Claude-3.5-Sonnet）
  • 一半在雨中追寻着迷失的自己（GPT-4 o）
  • 在城市的灯火下，影子交错编织梦想（qwen-1.5-110 b）
  • 夜色如墨，心弦却泛起涟漪（gemma-2-27 b）
  • 那是你，在梦里轻轻地呼唤（gemini-1.5-flash）
  • 我在迷茫的时刻，找到了归航（GPT-3.5-turbo）
